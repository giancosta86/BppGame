BppGame

ScalaFX app for the Bin Packing Problem

Run with MoonDeploy Download zip View on GitHub

BppGame is a ScalaFX application for interactively solving bin-packing problems (BPP); it is extensively based on TwoBinKernel, a library provided by the TwoBinPack suite.

BppGame is designed to be very simple and intuitive - but you can find further instructions in its practical wiki.
